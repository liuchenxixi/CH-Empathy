# CH-Empathy

## 项目简介

CH-Empathy 是一个致力于提升人工智能同理心和人文关怀能力的项目。

## 功能特性

- 🤖 AI 同理心模型
- 💬 自然对话交互
- 🎯 人性化响应
- 🌐 多语言支持

## 快速开始

### 环境要求

- Python 3.8+
- Git

### 安装步骤

```bash
# 克隆项目
git clone https://github.com/liuchenxixi/CH-Empathy.git

# 进入项目目录
cd CH-Empathy

# 安装依赖（如果有的话）
pip install -r requirements.txt
```

### 使用方法

```bash
# 运行项目
python main.py
```
